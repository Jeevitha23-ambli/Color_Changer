This is a simple interactive web project built using HTML, CSS, and JavaScript.

Features
Box 1: Changes background color to yellow and shows a random number when the mouse hovers.
Box 2: Cycles through different colors (red, blue, pink) and shows a random number.
Box 3: Generates a random RGB color and displays a random number.
Box 4: On click, all boxes change colors randomly and display random numbers.

Technologies Used
HTML
CSS
JavaScript (DOM Manipulation & Event Listeners)


Learning Concepts
DOM Manipulation
Event Listeners (mouseover, mouseleave, click)
Random number generation
Dynamic color changes using rgb()
